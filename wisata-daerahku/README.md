# Wisata Lombok - Project UTS Front-End Developer

Website statis sederhana menggunakan Bootstrap 5.

## Isi Proyek
- index.html (Beranda)
- destinasi.html (Daftar destinasi)
- detail.html (Detail destinasi, memakai query param ?dest=)
- galeri.html (Galeri foto)
- kontak.html (Form kontak & ulasan)
- assets/css/style.css
- assets/js/main.js
- assets/img/* (placeholder images)

## Cara Deploy
1. Extract folder `wisata-lombok`
2. Upload ke GitHub repository dan aktifkan GitHub Pages (branch `main` atau `gh-pages`)
   atau drag & drop ke Netlify.
3. Semua file sudah relatif; Bootstrap menggunakan CDN.

## Catatan
- Gambar placeholder dibuat otomatis. Ganti `assets/img` dengan foto asli bila perlu.
- Kalkulator estimasi ada di beranda; form kontak validasi client-side.

Dibuat untuk memenuhi soal UTS Front-End Developer.
